# AI Resume Analyzer & Interview Question Generator

AI-powered ATS Resume Screening and Interview Question Generation System using Streamlit and Mistral 7B.

## Features

* Resume Upload (PDF/DOCX)
* ATS Score Analysis
* Resume Selection Prediction
* Skill Matching
* Missing Skill Detection
* Resume Improvement Suggestions
* Interactive Skill Visualization
* AI-Powered Interview Question Generator
* Role-Based Question Generation
* Difficulty Level Selection
* Technical / HR / Behavioral Questions
* Download Questions as TXT File

## Technologies

* Python
* Streamlit
* Transformers
* PyTorch
* Mistral 7B Instruct v0.2
* Plotly
* Pandas
* PyPDF2
* python-docx
* BitsAndBytes

## Supported Job Roles

* Data Scientist
* Web Developer
* ML Engineer
* UI/UX Designer
* Frontend Developer
* Backend Developer
* DevOps Engineer
* Software Engineer
* Data Analyst


## Project Workflow

1. Upload Resume
2. Extract Resume Text
3. Analyze Skills
4. Generate ATS Score
5. Predict Resume Selection
6. Show Suggestions
7. Generate AI Interview Questions

## Model Used

* Mistral 7B Instruct v0.2
